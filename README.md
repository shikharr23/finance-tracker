# Finance Tracker

A small personal finance tracker app to record income and expenses, visualise trends, and manage transactions. Built as a learning project with a Vite/React frontend and a Node/Express backend.

Simple personal finance tracker (React + Node). This repo contains a frontend (Vite/React) and a backend (Node/Express). The app supports adding, editing, deleting transactions and shows charts on the dashboard.

## Stack

- Frontend: React, Vite, Tailwind CSS, Recharts, Axios
- Backend: Node.js, Express, MongoDB (Mongoose), JSON Web Tokens (JWT)
- Dev tools: Vite, ESLint, Prettier
- Deployment: Vercel / Netlify (frontend), Render / Railway (backend)
